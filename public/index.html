PDF Translate

Upload a PDF, translate each page to English, and get a downloadable version with the translated text gently overlaid on every page.

Download translated PDF

How it works

  • PDF.js extracts the text from each page inside your browser (no servers needed).
  • LibreTranslate sinks a translation request for every chunk of text and returns English copy.
  • pdf-lib draws the translation in a translucent callout so the original layout stays visible.
