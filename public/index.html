PDF Translate

Upload a PDF in any language, see a quick translation, and download an English version with the new text overlaid on each page.

Download translated PDF

How it works

  • The app extracts the text per page and translates it to English using an API.
  • The translated text is drawn in a lightly opaque box near the bottom of each page without altering the original artwork.
  • The output is a brand-new PDF you can download and share.
